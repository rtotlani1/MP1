MKS Ventures — Website
A professional static website built for MKS Ventures, a Delhi-based real estate development firm operating across residential and commercial segments.
Client
MKS Ventures (family business). The client is my father, who runs the firm. The goal was to establish a clean, credible web presence the firm can share with investors, partners, and clients.

What I Learned

How to structure a multi-page site with consistent navigation using semantic HTML 
How to link an external CSS stylesheet and keep styles organized in one place
How to use CSS Flexbox and Grid to build responsive layouts
How to make a full-screen hero background image using position: absolute and background-size: cover
How to use CSS custom properties (variables) to keep a consistent color palette across the whole site
How to think about a real client's needs before writing any code — the interview and proposal process shaped every design decision
